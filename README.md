# Система учёта электроинструмента на предприятии

Flask-приложение для учёта электроинструмента, выдачи сотрудникам и фиксации ремонта.

## Что где находится

### Техническое задание

- `docs/zadanie_1_tz.docx` - ТЗ по заданию 1.

### Серверная часть

- `app.py` - Flask-маршруты и обработчики HTTP-запросов.
- `database.py` - Функции работы с SQLite: создание таблиц, выборки, добавление, выдача, возврат, ремонт.

### Клиентская часть

- `templates/index.html` - HTML-шаблон главной страницы.
- `static/styles.css` - Оформление страницы, карточек, таблицы и кнопок.

### База данных

- `electrotools.db` - SQLite-база с демо-данными.
- `schema.sql` - SQL-схема таблиц.
- `database.py` - SQL-запросы и функции доступа к данным.

## Отдельные отчёты по заданиям

- `docs/zadanie_2_server.docx` - серверная часть.
- `docs/zadanie_3_client.docx` - клиентская часть.
- `docs/zadanie_4_database.docx` - база данных.

## Запуск

```powershell
pip install -r requirements.txt
python app.py
```
