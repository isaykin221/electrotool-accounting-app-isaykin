# Документация проекта

В этой папке лежат материалы для сдачи практических заданий 2-4 по теме 7.

## Файлы

- `otchet_zadaniya_2-4.md` - текст отчёта в формате Markdown.
- `otchet_zadaniya_2-4.docx` - тот же отчёт в формате Word.
- `screenshots/main.png` - скриншот главной страницы приложения.

## Что описано в отчёте

- серверная часть: структура файлов, маршруты, функции, примеры запросов и ответов;
- клиентская часть: основные блоки страницы, кнопки, формы, навигация и пользовательский сценарий;
- база данных: таблицы, поля, связи, SQL-запросы и примеры данных.
