# Документация проекта

Проект: Система учёта электроинструмента на предприятии.

## Файлы отчётов

- `zadanie_2_server.docx` - описание серверной части: структура, функции, параметры, ответы сервера.
- `zadanie_3_client.docx` - описание клиентской части: экран, кнопки, формы, сценарий пользователя.
- `zadanie_4_database.docx` - описание базы данных: таблицы, поля, связи, SQL-запросы и демо-данные.

Также оставлен общий отчёт `otchet_zadaniya_2-4.docx`, но для сдачи удобнее использовать три отдельных файла.
